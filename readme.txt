== Description ==

Wave Ya Flag - Show your opinion with your flag.  
This Widget shows a 3D-Flag of an Image that you want to show.  
::: Requires FlashPlayer 10 :::

Demo:<a href="http://flashapplications.de/?p=668" target="_blank">http://flashapplications.de/?p=668</a>

== Installation ==

1. Make sure you're running WordPress version 2.3 or better. It won't work with older versions.  
2. Download the zip file and extract the contents.  
3. Upload the 'wave-ya-flag' folder to your plugins directory (wp-content/plugins/).  
4. Activate the plugin through the 'plugins' page in WP.  
5. See 'Options->Wave Ya Flag' to adjust things like display size, etc...

= In order to actually display the 3D Flag, you have three options. =  
1. Create a page or post and type \[wp-flag] anywhere in the content. This 'tag' will be replaced by the flash movie when viewing the page. See [here\](http://flashapplications.de/?p=668) for more info.  
2. Add the following code anywhere in your theme to display the Flag. "<?php wp_flag(); ?>" This can be used to add Flag to your sidebar.  
3. The plugin adds a widget, so you can place it on your sidebar through 'Appearance'->'Widgets'. Open the widget to access it's own set of settings (background color,  etc).
